# StoneSpider
## 算法流程
### 获取屏幕截图
### 根据截图分析当前处于何种状态
### 若当前处于非游戏模式，则根据当前状态判断要进行哪些操作进入游戏模式
### 若当前属于游戏模式：
#### 判断是谁的回合
#### 识别当前费用
#### 识别当前手牌
##### 识别手牌随从牌
##### 识别手牌法术牌
#### 识别战场上我方随从
##### 识别战场上我方可以攻击的随从
#### 识别战场上敌方随从
##### 识别敌方嘲讽随从
##### 识别敌方高危险值随从
#### 识别敌方英雄血量
#### 根据已识别信息进行策略
